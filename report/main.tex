  \section*{Abstract}

  This intership aimed at building and caracterising a new method to generate a \(1.7\) GHz frequency shift of the \(589\) nm main laser in order to create a repumping beam. When imaging ultracold gases of Sodium atoms, a repumping beam is needed in order to depopulate blackened states which can't be imaged. The goal was to caracterize new methods to generate this \(1.7\) GHz shift using \(428\) MHz Acousto-Optic Modulators (AOM) in double-pass or four-pass configuration instead of the expensive and inefficient \(1.7\) GHz AOM currently used in the experiment.
